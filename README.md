Innovexa Digital Agency Landing Page
Overview

Innovexa is a creative digital agency landing page designed to showcase the services and values of a forward-thinking agency. This project emphasizes a modern dark mode design with bold typography, interactive elements, and responsive layouts, perfect for businesses looking to establish a professional online presence.

Key Features:

Responsive Layout: Ensures seamless display across mobile, tablet, and desktop devices.

Dark Mode Theme: Uses a sleek, dark color scheme with neon green accents to create a high‑end look.

Hero Section: Features a captivating hero section with key business metrics and a strong call-to-action button.

Services Section: Displays agency offerings in a clean, grid-based format for easy navigation.

Team & Testimonials: Provides credibility through team profiles and client reviews.

Interactive FAQ Section: Accordion-style dropdown for common queries to engage users.

Newsletter Signup: Keeps users in the loop with a simple subscription form.

Technologies Used:

HTML5 & CSS3 (Tailwind CSS / Custom CSS)

Font Awesome icons and Google Fonts for modern typography

How to Run:

Clone the repository.

Open the index.html file in your preferred browser.

Enjoy the smooth, responsive design of the Innovexa landing page!
